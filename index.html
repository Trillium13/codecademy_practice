HTML Table Reference

Table Elements

Tag Name Formatted Tag Usage
Table Element <table> Creates a two-dimensional table made of rows and columns
Table Row Element <tr> Adds rows to a table
Table Data Element <td> Adds a cell of data to a table
Table Body Element <tbody> Contains all table data other than the table heading and table footer content
Table Head Element <thead> Defines the headings of table column encapsulated in table rows
Table Footer Element <tfoot> Uses table rows to give footer content or summarize content at the end of a table
Table Heading Element <th> Adds titles to rows and columns of a table. Must be enclosed in a table row element

Table Attributes

Tag Name Formatted Tag Usage
colspan Attribute colspan="x" The cell will span 'x' number of columns up to 1000
rowspan Attribute rowspan="x" The cell will span 'x' number of rows up to 65534
